# 算法相关总结



将元素进行交换的方法:

```java
/**
     * 交换数组元素
     * 
     * @param arrays
     * @param a
     * @param b
     */
private static void swap(int[] arrays, int a, int b, boolean debug) {
    arrays[a] ^= arrays[b];
    arrays[b] ^= arrays[a];
    arrays[a] ^= arrays[b];
    log(true, "[" + arrays[b] + "] <-> [" + arrays[a] + "] >>> " + Arrays.toString(arrays));
}
```
