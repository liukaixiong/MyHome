# Spring 5的一些特性



![1526213716631](../../../image/wz_img/1526213716631.png)



- 依赖JDK8+和java EE7 + 以上版本，支持注解进行编程
- 新增函数式编程
- 支持使用REST断点执行反应式编程
- 支持HTTP 2.0
- 新增Kotlin和Spring Webflux可使用LamBda表达式注册Bean
- Spring WebMvc支持最新的API
- 使用JUnit5 执行条件和并发测试使用Spring WebFlux执行集成测试,核心容器优化
